# Flow.Launcher.Plugin.NJUPTWIFI

这个插件允许在南京邮电大学轻松使用 Flow Launcher 登录 WiFi。

## 使用方法

1. 在 Flow Launcher 设置中输入你的账号和密码。
2. 按下 Alt+空格，然后输入 "wifi" 来提示登录、注销或检查在线状态。按 Enter 完成 WiFi 的登录或注销操作以及查看是否在线操作。
